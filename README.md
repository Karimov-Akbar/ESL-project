# ESL Test Project: Custom Click/Hold Logic (PCA10059)

Этот проект реализует управление светодиодами nRF52840 Dongle (PCA10059) на основе логики "Короткий клик" и "Удержание" с использованием низкоуровневых драйверов GPIO HAL (Hardware Abstraction Layer) и циклических задержек `nrf_delay` для тайминга.

Проект скорректирован с учетом **фактического** расположения светодиодов, не совпадающего со стандартными названиями в nRF5 SDK.

## Логика работы

| Действие (Кнопка SW1) | Наблюдаемый Цвет | Физический Пин | Логика |
| :---: | :---: | :---: | :--- |
| **Короткий Клик** (50 мс < T < 500 мс) | **Красный** | P0.08 | Инвертирует состояние: включает/выключает Красный LED. |
| **Удержание** (T > 500 мс) | **Синий** и **Зеленый** | P1.09 и P0.12 | Запускает попеременное мигание Синего и Зеленого LED. Выключает Красный LED, если он был включен. |

## Технические детали

* **Целевая плата:** nRF52840 USB Dongle (PCA10059)
* **Среда:** nRF5 SDK (сборка через GCC Makefile)
* **Тайминг:** Используется циклический счетчик (`press_duration_ms`) с интервалом 5 мс (`nrf_delay_ms(5)`).

### Назначение пинов в коде (для разработчика)

| Название переменной | Фактический Пин | Физический Цвет (Наблюдаемый) | Роль |
| :--- | :---: | :---: | :--- |
| `LED_CLICK_RED` | P0.08 | Красный | LED для клика |
| `LED_HOLD_BLUE` | P1.09 | Синий | Первый LED для удержания |
| `LED_HOLD_GREEN` | P0.12 | Зеленый | Второй LED для удержания |
